if you want to make the data into mysql:

1) create a virtual environment
    virtualenv venv
2) navigate to mytrisul->venv->Scripts
3) enter activate
4) come back to mytrisul directory
5) install mysql client
6) create a database named trisul in your mysql
7) py manage.py makemigrations
8) py manage.py migrate
9) py manage.py runserver
